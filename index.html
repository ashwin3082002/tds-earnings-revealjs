Sample Analytics Snippet

Example: computing quarterly Net Interest Margin (NIM) using Python.


def compute_nim(interest_income, interest_expense, earning_assets):
    """
    Net Interest Margin (NIM) = 
    (Interest Income - Interest Expense) / Average Earning Assets
    """
    net_interest = interest_income - interest_expense
    return net_interest / earning_assets

# Example numbers in millions
interest_income = 1250.0
interest_expense = 780.0
earning_assets = 18500.0

nim = compute_nim(interest_income, interest_expense, earning_assets)
print(f"Quarterly NIM: {nim:.2%}")

Key Financial Formulas

**Return on Equity (ROE)**:

\( \displaystyle ROE = \frac{\text{Net Income}}{\text{Average Shareholders' Equity}} \)

**Cost of Equity (CAPM)**:

\( \displaystyle k_e = r_f + \beta \left( E[R_m] - r_f \right) \)

**Loan Loss Coverage Ratio**:

\( \displaystyle \text{Coverage} = \frac{\text{Loan Loss Provisions}}{\text{Gross NPA}} \)
